# AI-BASED-PERSONALIZED-SHOPPING-PLATFORM
This project is a **Flask-based AI application** that provides fashion and makeup recommendations using image analysis.   The system detects:  ✔ Skin Tone   ✔ Body Type   ✔ Outfit Recommendations   ✔ Makeup Suggestions   ✔ Color Palette   ✔ Camera Capture + Image Upload Support    It is built for **B.Tech Final Year Project**
